# Site web de l’église Saint-Jean-Baptiste d’Aulès – Doazit (Landes)

Ce projet vise à créer un site web moderne pour valoriser l’église Saint-Jean-Baptiste d’Aulès, située à Doazit dans les Landes. Il permet de découvrir son histoire, suivre ses actualités, et soutenir sa préservation grâce à un appel aux dons.

Ce dépôt est un **monorepo** basé sur le **Launchpad officiel de Strapi**, combinant un back-end Strapi et un front-end Next.js.

---

## Objectifs

- Mettre en valeur le patrimoine architectural et historique de l’église.
- Publier des articles, événements et témoignages via un blog.
- Faciliter les dons pour les travaux de restauration.
- Créer un site accessible, rapide et responsive.

---

## Fonctionnalités

- Page d’accueil avec présentation de l’église
- Blog dynamique alimenté via Strapi
- Galerie photo
- Page de dons intégrée avec HelloAsso
- Formulaire de contact
- SEO optimisé

---

## Technologies utilisées

| Outil / Framework | Rôle                                      |
| ----------------- | ----------------------------------------- |
| Strapi            | CMS headless pour la gestion des contenus |
| Next.js           | Front-end React avec SSR                  |
| Yarn Workspaces   | Monorepo pour gérer les dépendances       |
| HelloAsso         | Plateforme de dons pour associations      |
| TypeScript        | Typage statique                           |
| Tailwind CSS      | Design responsive                         |
